# StartJAVA

* Write once, run anywhere

* Environment: Ubuntu 14.04

* IDE: IntelliJ IDEA
	* IDEA is a smart Java IDE produced by JetBrains, a Czech Company.
	* It is similar to eclipse but more intelligent.
	* /src: source files .java
